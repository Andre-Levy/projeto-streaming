# 🎬 Projeto Streaming - Lumen

## 📌 Sobre o Projeto
O **Lumen** é um projeto fictício de plataforma de streaming desenvolvido com o objetivo de simular uma aplicação moderna de consumo de conteúdo digital, como filmes e séries.

Este projeto foi criado para fins de portfólio, com foco em demonstrar habilidades em desenvolvimento **Front-End**, design de interfaces e experiência do usuário (UI/UX).

---

## 🚀 Funcionalidades
- 🎥 Catálogo de filmes e séries  
- 🔍 Sistema de navegação intuitivo  
- 🖱️ Efeitos interativos ao passar o mouse  
- 📱 Layout totalmente responsivo  
- 🎨 Design moderno inspirado em plataformas reais de streaming

## 💡 Objetivo
O principal objetivo do projeto é simular uma plataforma real de streaming, colocando em prática conceitos como:

- Estruturação de layouts responsivos  
- Organização de código  
- Criação de interfaces modernas  
- Experiência do usuário
